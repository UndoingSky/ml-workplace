# Student Marks Analyzer

This project is a small NumPy practice notebook that generates a random set of student marks and analyzes them with basic array operations.

## What It Does

The notebook:

- creates a $20 \times 4$ random dataset of marks from 1 to 100
- calculates each student's average score
- calculates the average score for each subject
- finds the highest and lowest score in each subject
- reshapes the dataset and splits it into smaller arrays
- merges the split arrays back together with `vstack`
- appends, deletes, and inserts rows in the dataset

## Files

- `student_marks_analyzer.ipynb` - the main notebook

## Requirements

- Python 3
- NumPy
- Jupyter Notebook or VS Code with notebook support

## How To Run

1. Open `student_marks_analyzer.ipynb` in VS Code or Jupyter.
2. Run the notebook cells from top to bottom.
3. Review the printed output to see the generated marks and analysis results.

## Notes

- The dataset is generated randomly each time the notebook runs, so the results will change on every execution.
- This notebook is intended as a learning exercise for NumPy indexing, reshaping, and aggregation.
